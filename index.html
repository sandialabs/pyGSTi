The pyGSTi website is being rebuilt!

pyGSTi is an open-source software for modeling and characterizing noisy quantum information processors (QIPs), i.e., systems of one or more qubits.

Currently (as of March 2022) the pyGSTi website is undergoing a redesign; here are some relevant pages in the meantime:

  • pyGSTi on GitHub, if you want to install pyGSTi from source.
  • pyGSTi on PyPI, if you want to install pyGSTi via pip.
  • The Quantum Performance Laboratory, the research group at Sandia National Laboratories that develops and maintains pyGSTi. (We're currently looking to hire!)
  • "Intro to PyGSTi", an introductory talk demonstrating common pyGSTi use cases, given as part of the Unitary Fund Quantum Software Talk series.
